## 1. システム概要
本プロジェクトは、以下の3つの主要コンポーネントから構成されています。
1. **`AD7606-sensor-servo.ino` (Arduino)**
   * 24bit/16bit高精度ADC「AD7606」をSPI通信（1MHz）で制御し、6軸力覚センサなどの生データを1kHz（1ms間隔）で高速サンプリング。
   * 環境センサ「BME280」による大気圧・温度・湿度の取得（I2Cバスのハングアップ自動リセット機能付き）。
   * ターゲット角度へ滑らかに追従するサーボモータ制御。
2. **`log_csv_AD7606.py` (Python / リアルタイムロガー)**
   * PySide6およびpyqtgraphを採用した高応答なリアルタイムUI。
   * センサーデータから空気密度（$\rho$）および動圧（$q$）をリアルタイム演算し、揚力係数（CL）および抗力係数（CD）を算出。
   * ポテンショメータの値を監視し、目標角度へ自動調定する「3段階フィードバック制御（大移動・微調整・最終確認）」を搭載。
3. **`analyzer_integration.py` (Python / データアナライザー)**
- 本アナライザーは、以下の2つの主要なアプローチをタブ操作で切り替えて実行できます。
  - 📊 全体表示タブ (複数データセット比較):
  異なる風洞試験データセット（複数ケースや迎角掃引データ）のサマリーを一元管理し、 $C_L$ 曲線、 $C_D$ 曲線、およびドラッグ極線（Drag Polar）を重ね描き比較します。
  - 📈 時系列解析タブ (詳細信号解析):
  特定のデータセットおよび試験角度をプルダウンで指定し、サンプリング時間軸に沿った動的な挙動、3軸力/3軸モーメントの変化、風圧中心の位置ズレ、空気力学的振動の周波数成分をミリ秒単位で詳細に追跡します。

## 2. ハードウェア構成とピンアサイン
本システムは **Arduino (Uno/Megaなど5V系動作のマイコンボード)** をベースに、レベル変換IC（FXMA108等）を介して各センサと接続することを想定しています。

### AD7606 (SPI接続)
| AD7606 ピン | Arduino ピン | 備考 |
| :--- | :--- | :--- |
| CS | D10 | チップセレクト |
| SDI (MOSI) | D11 | データ入力 |
| DOUTA (MISO)| D12 | データ出力 |
| SCLK (SCK) | D13 | クロック |
| CONVST | D9 | 変換開始パルス |
| RESET | D8 | ハードウェアリセット |
| BUSY | D7 | 変換完了監視 |

### その他のデバイス
* **BME280**: I2C接続 (SDA / SCL) ※アドレス `0x76`
* **RCサーボモータ**: D6 (PWM/パルス制御)
* **ポテンショメータ**: AD7606のCH7（アナログ7ch）に入力
* **風速計（アナログ電圧出力）**: AD7606のCH8（アナログ8ch）に入力

## 4. アプリ使用方法
## 4.1. 計測アプリ
1. ArduinoをPCにUSB接続します。
2. `log_csv_AD7606.py` を実行します。
```bash
# uvを使用する場合
uv run log_csv_AD7606.py

#uvを使用しない場合
python log_csv_AD7606.py
```
3. ポートを選択して Connect をクリック。
4. 模型の風力風袋を引く場合は Update Zero で6軸ゼロ点補正（Tare）を行います。
5. セール面積（デフォルト: $0.1875\,\text{m}^2$）や計測タイマーを設定し、Start REC で記録を開始します。角度変更時は Feedback モードで目標迎角を入力し Move を押すことで、自動で迎角が調定され、ファイル名に自動で角度が付与されたCSVが生成されます。
 
## 4.2 解析アプリ
1. `analyzer_integration.py` を実行します。
```bash
# uvを使用する場合
uv run analyzer_integration.py

#uvを使用しない場合
python analyzer_integration.py
```
2. Load From Folder または Load From Archive から、計測データが格納されたフォルダ（またはZIP/7zファイル）を選択します。
3. 規則的に命名されたCSV（例: ..._test_0.0.csv, ..._test_5.0.csv）から自動で角度ごとのCL/CDが算出され、データセット間で重ね描き比較が行えます。
 
1. 各タブの機能詳細

### **📊 全体表示タブ (複数データセット比較)**

- データセット登録:<br>
フォルダ選択ダイアログまたはアーカイブ（.zip / .7z）選択経由で、複数ケースを読み込むと、それらが個別に色付けされて管理リストに追加されます。

- 自動ペアリング:<br>
同一データセット内で角度（Angle）ごとの tare と test ファイルを自動検出し、4.1〜4.3項の数式に基づいて全角度の平均 $C_L$、$C_D$、および $L/D$ を算出してデータベース化します。

- スイープ曲線の自動描画:<br>
   CL Curve: 迎角 $\alpha$ に対する $C_L$ の推移を描画。<br>
   CD Curve: 迎角 $\alpha$ に対する $C_D$ の推移を描画。<br>
   Drag Polar: 抗力 $C_D$ に対する揚力 $C_L$ の二次元極線を描画。<br>

### **📈 時系列解析タブ (詳細信号解析)**
- プルダウン同期:<br>
全体表示タブに読み込まれたデータセット名と、その中に含まれる有効なテスト角度（例: 0.0, 5.0, 10.0 度など）が動的にプルダウンメニューへ連動・反映されます。

- 生データ・フィルタプロット:<br>
生データの瞬時値（薄い線）と、選択したノイズ除去フィルタ（SMA / EMA）処理後の値（太い実線）をタイムスタンプに沿ってリアルタイムで重ね描きプロットします。

- 双方向連携式レンジセレクト:<br>
グラフ上のスライダー（グレーの選択バー）をマウスで直感的に伸縮・移動させることで、詳細解析の評価対象区間を自由に調整できます。

- CoP 2Dマッピング:<br>
評価対象区間内のすべての風圧中心位置 $(X_{\text{CoP}}, Z_{\text{CoP}})$ のばらつきを二次元散布図としてマッピングし、その空間平均（重心位置）をターゲットマークで描画して、セイルの空力安定性を一目で視覚化します。

- レポート出力:<br>
選択範囲内の詳細な時系列CSVデータ、または各パラメータの「平均値」「標準偏差（SD）」「RMS値」「最大/最小」を出力したテキスト形式のサマリーレポートをワンクリックでエクスポート可能です。

## 5. 入力データの仕様と命名規則
### 5.1 フォルダ構造およびファイル名
- ロード対象のフォルダまたは圧縮ファイル（.zip / .7z）内には、以下の命名規則に合致するCSVファイル群が含まれている必要があります。
- **ファイル名規則**:
`{任意の文字列}_{シーン名(TareまたはTest)}_{角度(実数/整数)}.csv`

例:<br>
`case01_tare_10.0.csv`  # 迎角10.0度における風速0の時のゼロ点測定データ (Tare)
`case01_test_10.0.csv`  # 迎角10.0度における風洞本試験データ (Test)

### 5.2 CSVファイルのカラムヘッダー
各CSVファイルは、以下のカラム（1行目にヘッダー）を含むカンマ区切りテキストである必要があります。
```
| カラム名 | 物理的な意味と単位 |
| Timestamp | 計測日時（YYYY-MM-DD HH:MM:SS.fff 形式など） |
| Fx(N) | 風下方向（ドラッグ方向）の測定力 $[N]$ |
| Fy(N) | 風上から見て右向き（リフト方向）の測定力 $[N]$ |
| Fz(N) | 下向きの測定力 $[N]$ |
| Mx(Nm) | ロールモーメント $[N \cdot m]$ |
| My(Nm) | ピッチモーメント $[N \cdot m]$ |
| Mz(Nm) | ヨーモーメント $[N \cdot m]$ |
| AvgWind | 測定部代表風速 $[\text{m/s}]$ |
| Temperature | 測定部乾球温度 $[^\circ\text{C}]$ |
| Humidity | 測定部相対湿度 $[\%]$ |
| Pressure | 測定部気圧 $[\text{hPa}]$ |
```

💡 注: CSVの先頭行に # で始まるメタデータ（例: # Sail Area: 0.1875, # Memo: ...）が存在する場合、システムは代表面積等の初期設定値をヘッダーから自動的に抽出します。

## 6. 物理モデルと計算アルゴリズム
### 6.1 湿り空気密度の算出
空気の乾燥状態および湿潤状態を反映した高精度な状態方程式（湿り空気の状態方程式）を用いて、刻々と変化する空気密度 $\rho \ [\text{kg/m}^3]$ を算出します。

- (a) 飽和水蒸気圧 $E_s$ の計算（Tetensの式）<br>
乾球温度 $T \ [^\circ\text{C}]$ における飽和水蒸気圧 $E_s \ [\text{hPa}]$ を算出します。

$$E_s = 6.1078 \times 10^{\frac{7.5 \times T}{T + 237.3}}$$

- (b) 水蒸気分圧 $P_v$ の計算<br>
相対湿度 $H \ [\%]$ から水蒸気分圧 $P_v \ [\text{hPa}]$ を算出します。

$$P_v = E_s \times \frac{H}{100}$$

- (c) 乾き空気分圧 $P_d$ の計算<br>
測定全圧（大気圧） $P \ [\text{hPa}]$ から水蒸気分を差し引き、乾燥空気分圧 $P_d \ [\text{hPa}]$ を算出します。

$$P_d = P - P_v$$

- (d) 湿り空気密度 $\rho$ の合成式<br>
絶対温度 $T_k \ [\text{K}] = T + 273.15$ を用いて、乾燥空気（気体定数 $R_d = 287.05 \ \text{J/(kg}\cdot\text{K)}$）と水蒸気（気体定数 $R_v = 461.495 \ \text{J/(kg}\cdot\text{K)}$）の密度を合計します。

$$\rho = \frac{P_d \times 100}{R_d \times T_k} + \frac{P_v \times 100}{R_v \times T_k}$$

※分圧の値を $100$ 倍することで $\text{hPa}$ から $\text{Pa} \ (\text{N/m}^2)$ に変換しています。

### 6.2 動圧の算出

- 空気密度 $\rho$ と代表風速 $V \ [\text{m/s}]$（AvgWind）から、動圧 $q \ [\text{Pa}]$ を算出します。

$$q = \frac{1}{2} \rho V^2$$

### 6.3 空力係数 ($C_L, C_D$) および揚抗比 ($L/D$) <a id="section-63"></a>

代表面積（セイル等） $S \ [\text{m}^2]$ を用い、測定された無次元性能係数を算出します。

- (a) 揚力係数 (Lift Coefficient) $C_L$

$$C_L = \frac{F_{y,\text{net}}}{q \times S}$$

※天秤の設定に基づき、風下に対して直交する側方力 $F_{y,\text{net}}$ を揚力成分として取り扱います。

- (b) 抗力係数 (Drag Coefficient) $C_D$

$$C_D = \frac{F_{x,\text{net}}}{q \times S}$$

※風下方向力 $F_{x,\text{net}}$ を抗力（ドラッグ）成分として取り扱います。

- (c) 揚抗比 (Lift-to-Drag Ratio) $L/D$<br>
空気力学的な高効率指標である揚抗比を算出します。

$$\frac{L}{D} = \frac{C_L}{C_D}$$

### 6.4 ゼロ点（Tare）補正

風洞の送風オフ状態（風速 $V = 0$）における天秤および供試体の自重や初期歪みなどを「Tare（ゼロ点）」として除去します。同一角度 $\theta$ で測定されたTareデータの平均値を求め、Testデータから減算します。

$$\begin{aligned}
F_{x,\text{net}}(t) &= F_{x,\text{test}}(t) - \overline{F_{x,\text{tare}}} \\
F_{y,\text{net}}(t) &= F_{y,\text{test}}(t) - \overline{F_{y,\text{tare}}} \\
F_{z,\text{net}}(t) &= F_{z,\text{test}}(t) - \overline{F_{z,\text{tare}}} \\
M_{x,\text{net}}(t) &= M_{x,\text{test}}(t) - \overline{M_{x,\text{tare}}} \\
M_{y,\text{net}}(t) &= M_{y,\text{test}}(t) - \overline{M_{y,\text{tare}}} \\
M_{z,\text{net}}(t) &= M_{z,\text{test}}(t) - \overline{M_{z,\text{tare}}}
\end{aligned}$$

### 6.5 風圧中心 (Center of Pressure: CoP)

流体力が供試体に作用する等価な中心位置（マストや支持軸を原点とした前後方向 $X_{\text{CoP}} \ [\text{m}]$ および高さ方向 $Z_{\text{CoP}} \ [\text{m}]$）をモーメントと力の釣り合いから決定します。

- (a) 高さ方向風圧中心 $Z_{\text{CoP}}$ の算出<br>
ロールモーメント $M_{x,\text{net}}$ と右向き力 $F_{y,\text{net}}$ の関係から、高さを算出します。

$$Z_{\text{CoP}} = -\frac{M_{x,\text{net}}}{F_{y,\text{net}}}$$

- (b) 前後方向風圧中心 $X_{\text{CoP}}$ の算出<br>
ヨーモーメント $M_{z,\text{net}}$ と右向き力 $F_{y,\text{net}}$ の関係から、前後位置を算出します。

$$X_{\text{CoP}} = \frac{M_{z,\text{net}}}{F_{y,\text{net}}}$$
⚠️ 特異点対策（発散抑制）:
$\left|F_{y,\text{net}}\right| \le 0.01 \ [\text{N}]$ となる瞬間（揚力が極小、またはほぼゼロの場合）は分母が極端に小さくなり、算出値が急激に発散します。これを防ぐため、システム内部で当該フレームを NaN（非数）としてマスクし、前後の有効値で時間軸に沿って補間（Ffill/Bfill）します。

### 6.6 デトレンド処理と離散フーリエ変換 (FFT)

時系列データにおける周期的な空気力変動（フラッター現象や剥離渦によるカルマン渦の放出など）を捉えるため、ユーザーが指定した時間窓内で高速フーリエ変換 (FFT) を行います。

- (a) デトレンド処理 (Detrending)<br>
時系列データの直流（DC）成分（平均値 $\mu$）を除去し、純粋な交流変動成分 $y_{\text{detrend}}(t)$ のみを抽出します。

$$y_{\text{detrend}}(t) = y(t) - \frac{1}{N} \sum_{i=1}^{N} y(t_i)$$

- (b) 片側振幅スペクトル (Single-Sided Amplitude Spectrum)<br>
全サンプリング数 $N$ および時間解像度 $dt \ [\text{s}]$ から、実数値離散フーリエ変換を行い、物理的に直感的なスケールとなる片側振幅スペクトル $A(f)$ にスケーリングします。

$$A(f) = \frac{2}{N} \left| \mathcal{F} \{ y_{\text{detrend}} \} \right|$$

※ピーク周波数 $f_{\text{peak}} = \arg\max_{f > 0} (A(f))$ を自動検知し、構造物やセイルの共振解析用に出力します。

### 6.7 ノイズ低減フィルタ（移動平均・EMA）

測定データの高周波微小ノイズ（天秤の微小振動や気流の乱れ）を滑らかにするため、ユーザー設定の窓幅 $W \ [\text{pts}]$ によるデジタルフィルタを適用します。

- (a) 移動平均フィルタ (Simple Moving Average: SMA)

$$y_{\text{filtered}}(t_i) = \frac{1}{W} \sum_{k = -\lfloor W/2 \rfloor}^{\lfloor W/2 \rfloor} y(t_{i+k})$$

- (b) 指数移動平均フィルタ (Exponential Moving Average: EMA)

$$y_{\text{filtered}}(t_i) = \alpha \cdot y(t_i) + (1 - \alpha) \cdot y_{\text{filtered}}(t_{i-1})$$

$$\alpha = \frac{2}{W + 1}$$
